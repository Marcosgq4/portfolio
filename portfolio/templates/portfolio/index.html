Task Manager

This is my Task Manager App. This project was my final project to obtain the CS50W certification. The idea behind it was to create an app that could help the user prioritize their tasks. When you create a task for yourself, the program asks you to give it a due date and a level of importance. Then, it will show you all of your tasks, in order, with the most important tasks that are due sooner all the way at the top and the least important tasks that are due later, at the bottom. This would help anyone organyze their work day to accomplish the most important tasks in order of priority. You're also able to complete, update or delete your tasks at any point, if you choose to.

Network

This is my Network App. This project had the purpose to create an app resembling a social network. While this app is not as sophisticated as Facebook, it allows every user to create and edit their own posts. These posts are then posted according to their date of creation and paginated so that only a limited amount of them can be shown in each page. You can also follow or unfollow other users, as well as like or unlike their posts, and you can selectively see the posts of the users you follow, rather than watching everything available in the network.
